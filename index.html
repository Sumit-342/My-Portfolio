02

Resume Analyzer — AI-Powered Resume Evaluation Tool

Resume Analyzer is a web-based application that analyzes resumes and provides role-based insights. The system extracts key information such as name, email, phone number, LinkedIn, and GitHub from PDF resumes. It uses predefined skill datasets and matching logic to evaluate resumes for roles like AI/ML, Data Science, and Web Development. The application generates a resume score, predicts the best-fit role, highlights matched and missing skills, and provides actionable suggestions for improvement. It also includes a basic ATS-style evaluation and visual insights to enhance understanding. Currently, the system runs using Streamlit with a focus on clean UI and efficient analysis logic.
Future Goal: Enhancing the system using machine learning models for smarter skill matching and more accurate role prediction.

Python, Streamlit, PyMuPDF (Fitz), Regular Expressions (Regex), Pandas, Chart.js, Machine Learning Concepts

Live Project Github Repository

03

Lucy - Emotion Detection AI Assistant

Lucy is a personal offline AI assistant that detects human emotions from text and emojis. The system combines Natural Language Processing (NLP) with machine learning models to classify emotions such as Happy, Sad, and Angry. It processes user input, extracts features using TF-IDF, and predicts emotions using trained models. Additionally, emoji-based emotion mapping is used to improve prediction accuracy. Currently, Lucy runs locally in a terminal-based environment, focusing on core AI logic and performance.
Future Goal: Converting Lucy into a full AI assistant with a user interface and voice interaction.

Python, NLP, TF-IDF Vectorization, Scikit-learn, Machine Learning
